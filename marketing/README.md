# Marketing Materials — Enterprise Training

This folder contains all enterprise sales and marketing materials for the
**"From Vibes to Production"** training program.

## Files

| File | Purpose |
|---|---|
| `one-pager.md` | Single-page sales leave-behind / email attachment |
| `sales-deck-outline.md` | Slide-by-slide outline for enterprise sales pitch |
| `email-sequences.md` | Cold outreach + follow-up email templates |
| `linkedin-posts.md` | Organic LinkedIn content for awareness |
| `objection-handling.md` | Common objections + responses |
| `pricing-tiers.md` | Pricing structure and packaging options |
| `ideal-customer-profile.md` | ICP definition and qualification criteria |

## Positioning

**Program name:** From Vibes to Production  
**Category:** Enterprise AI Operationalization Training  
**Delivery:** Workshop (half-day / full-day) + self-paced online access  
**Audience:** Engineering teams, product organizations, technical leadership

## Core Message

> Most companies have given their teams access to AI tools.  
> Very few have given them the knowledge to use them in production.  
> This training closes that gap.
